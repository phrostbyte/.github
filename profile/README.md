PhrostByte is built by TensorSpace, Inc. for teams that need auditable security operations software they can run, inspect, and extend.

## What We Build

- Cross-platform endpoint agents for Linux, macOS, and Windows.
- A security operations dashboard for alert triage, simulations, response workflows, and security posture views.
- Replay and integration tooling for validating agent telemetry against dashboard behavior.
- AGPL-licensed public code in [phrostbyte/phrostbyte](https://github.com/phrostbyte/phrostbyte).

## Project Principles

- Transparent implementation over opaque black boxes.
- Practical automation that keeps analysts in control.
- Strong local validation and CI before release.
- Clear ownership by TensorSpace, Inc.
